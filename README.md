
# Lumina Digital - Agence de Transformation Numérique

Ce projet est une application web React (TypeScript + Tailwind CSS) représentant le site professionnel d'une agence fictive de conseil numérique.

## Structure du Projet

- `index.tsx` & `App.tsx` : Points d'entrée de l'application.
- `components/` : Dossier contenant les sections modulaires de la page d'accueil.
- `constants.tsx` & `types.ts` : Centralisation des données et types pour une meilleure maintenance.
- `index.html` : Fichier principal chargeant Tailwind via CDN pour une légèreté maximale.

## Fonctionnalités

- **Mobile-First & Responsive** : S'adapte parfaitement aux mobiles, tablettes et ordinateurs.
- **Hero Section** : Répond immédiatement aux questions "Qui sommes-nous ?", "Que faisons-nous ?" et "Comment aidons-nous nos clients ?".
- **Navigation Sticky** : Barre de menu qui reste accessible au scroll avec un effet de verre dépoli.
- **Performance** : Utilisation de Tailwind CSS pour un rendu rapide et des animations fluides.
- **Typographie Moderne** : Utilisation de "Inter" et "Plus Jakarta Sans" pour un look premium.

## Comment Utiliser

1. Téléchargez le dossier compressé.
2. Assurez-vous d'avoir un environnement React standard (Vite, Create React App ou autre).
3. Ce code est conçu pour fonctionner directement dans un environnement web moderne.

## À Propos de l'Entreprise (Fictive)

**Nom** : Lumina Digital
**Secteur** : Agence de transformation numérique (B2B).
**Cible** : PME et startups cherchant à moderniser leurs outils et augmenter leur ROI numérique.
